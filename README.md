# Tic Tac Toe - Java TDD #

## Getting started ##
[![Build Status](https://travis-ci.org/hanster/ttt-java.svg?branch=master)](https://travis-ci.org/hanster/ttt-java)

* Maven 3

### Build and Run ###
#### Maven ####

    java -jar target/ttt-1.0-SNAPSHOT-jar-with-dependencies.jar
